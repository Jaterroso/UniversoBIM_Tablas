# UniversoBIM Tablas

<img src="images/UniBimTablas_1.4.jpg" width="640" />

Herramientas para trabajar con Tablas en Revit. 

Creación automática de tablas y exportación a Excel

## Requisitos
* Autodesk Revit 2017
* Autodesk Revit 2018
* Autodesk Revit 2019
* Autodesk Revit 2020
* Autodesk Revit 2021
